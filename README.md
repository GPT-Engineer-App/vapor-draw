# vapor-draw

Create an app that shows a full screen drawing canvas. The user should be able to select a colour and brush size from a settings modal. Make it have a vaporware aesthetic.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/vapor-draw.git
cd vapor-draw
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
